### Getting started

 1. Clone project
 	
 	```
    git clone https://github.com/jzlin/dotnetcore-angular.git
    ```
    
 2. Restore the .NET and NPM dependencies.
 	
    ```
    cd dotnetcore-angular
    npm install
    ```
    
 3. Start local server. ([localhost:5000](http://localhost:5000))
 	
    ```
    npm start
    ```
